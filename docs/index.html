FlowDialog

v2.0.0

A framework-agnostic modal dialog library with flow/wizard support, dynamic content resizing, and beautiful animations.

View on GitHub Try Live Demo View Examples

Features

Framework-Agnostic

Pure vanilla JavaScript core with zero dependencies. Use it anywhere!

Flow/Wizard Support

Create multi-step dialogs with seamless transitions between steps.

Dynamic Resizing NEW

Automatically animate height changes as content grows or shrinks.

Beautiful Animations

Smooth CSS and JavaScript animations for opening, closing, and flowing.

jQuery Adapter

100% backward compatible with v1.x jQuery plugin API.

Angular Wrapper

Native Angular directive and service for seamless integration.

Keyboard Navigation

ESC to close, arrow keys for navigation, full accessibility support.

Responsive

Works beautifully on all screen sizes and devices.

TypeScript Support

Full TypeScript definitions included for better developer experience.

Quick Start

Vanilla JavaScript

// Create a dialog from an element
var dialog = new FlowDialog(document.getElementById('myDialog'), {
    width: 600,
    closeOnEscape: true,
    growToHeight: true,  // Enable dynamic resizing
    height: 500           // Max height when growToHeight is true
});

// Open the dialog
dialog.open();

// Listen to events
dialog.on('onOpen', function() {
    console.log('Dialog opened!');
});

jQuery (Backward Compatible)

// Initialize and open
$('#myDialog').flowdialog({ 
    width: 600,
    growToHeight: true
}).flowdialog('open');

// Events work the same way
$('#myDialog').on('flowdialog_onOpen', function() {
    console.log('Dialog opened!');
});

Angular

<!-- Declarative approach with directive -->
<div flow-dialog 
     flow-dialog-instance="myDialog"
     flow-dialog-width="600"
     title="My Dialog">
    <p>Content here</p>
</div>

<button ng-click="myDialog.open()">Open</button>

API Reference

Options

Option Type Default Description
height number | 'auto' 'auto' Height of the dialog content. If 'auto', height grows with content.
growToHeight Featured boolean false When true, the 'height' option becomes max-height, and content animates as it grows/shrinks.
width number 600 Width of the dialog in pixels.
showCloseButton boolean true Show the close button in the upper right corner.
closeOnEscape boolean false Allow closing the dialog by pressing ESC.
closeOnOverlayClick boolean false Allow closing by clicking outside the dialog.
autoOpen boolean false Automatically open the dialog on initialization.
useTransitions boolean true Enable animations when opening, closing, or resizing.
animateDuration number 250 Animation duration in milliseconds.
flow Array [] Array of elements or objects defining additional flow steps.
appendTo Element document.body Element to append the dialog to.
hideEmptyFooter boolean true Hide footer if no footer content is defined.
hideEmptyTitle boolean false Hide title bar if no title is defined and close button is hidden.

Methods

Method Arguments Description
open() index (optional) Opens the dialog. If index provided, opens to that flow step.
close() - Closes the dialog.
flow() command, ...args Control flow navigation. Commands: 'next', 'prev', 'index', or step number.
option() name, value (optional) Get or set an option value.
reposition() - Recenter the dialog on the page.
on() event, callback Attach an event listener.
off() event, callback (optional) Remove an event listener.
destroy() - Destroy the dialog instance and clean up.

Events

Event Description
onOpen Triggered when the dialog is opened.
onClose Triggered when the dialog is closed.
onPreClose Triggered before closing. Prevent default to cancel close.
onFlow Triggered when flow navigation occurs.
onReposition Triggered when dialog is repositioned.

Code Examples

Creating a Flow (Wizard)

// HTML structure
<div id="step1" title="Step 1">
    <p>Content for step 1</p>
    <div data-type="footer">
        <button onclick="dialog.flow('next')">Next</button>
    </div>
</div>

<div id="step2" title="Step 2" style="display:none">
    <p>Content for step 2</p>
    <div data-type="footer">
        <button onclick="dialog.flow('prev')">Previous</button>
        <button onclick="dialog.close()">Finish</button>
    </div>
</div>

// JavaScript
var dialog = new FlowDialog(document.getElementById('step1'), {
    width: 600,
    flow: [
        document.getElementById('step2')
    ]
});

Dynamic Content with growToHeight

var dialog = new FlowDialog(document.getElementById('myDialog'), {
    width: 600,
    growToHeight: true,    // Enable dynamic resizing
    height: 500,             // This becomes max-height
    useTransitions: true,  // Animate the resize
    animateDuration: 300    // Animation speed
});

dialog.open();

// When content changes, the dialog will automatically
// animate to the new height (up to max of 500px)

Event Handling

var dialog = new FlowDialog(element, options);

// Listen to events
dialog.on('onOpen', function() {
    console.log('Dialog opened');
});

dialog.on('onFlow', function() {
    console.log('Current flow index:', dialog.flowIndex);
});

dialog.on('onPreClose', function(e) {
    if (!confirm('Are you sure?')) {
        e.preventDefault(); // Cancel the close
    }
});
